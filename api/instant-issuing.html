​Instant Issuing Request

POST
/instantissuing

Create a new card instantly with custom settings and configurations.

​Headers

  • X-API-Key
    string
    required

    API key for authentication

  • Content-Type
    string
    required

    Content type of the request

​Request Body

  • cardholderName
    string

    Full name of the cardholder.

  • cardProduct
    string

    Type of card product.

  • deliveryMethod
    string

    Method of card delivery.

  • expiryMonths
    integer
    optional

    Number of months until expiry.

  • initialStatus
    string
    optional

    Initial status of the card.

​Responses

    • success
      boolean

      Indicates if the operation was successful.

    • cardDetails
      object
