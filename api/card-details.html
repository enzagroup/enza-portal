Set Card Delivery Address

POST
https://uat-api.enza.cloud/TX/v1
/cards/delivery-address

Update or set the card delivery address for a specific card. Used when the cardholder requests to receive a card at a different address than the default. All address fields are optional except cardId and countryId.

Set Card Delivery Address › Headers

​string · required

Content-Type
