Customer Management

Endpointhttps://uat-api.enza.cloud/TX/v1

Retrieve and manage customer information

Create New Customer

POST
https://uat-api.enza.cloud/TX/v1
/customers/create

Create a new customer (subject) in the system.

Process:

  1. Check if customer already exists
  2. Create customer with provided details
  3. Return success confirmation

Check Customer Existence

POST
https://uat-api.enza.cloud/TX/v1
/customers/search

Check if a customer exists in the system or not.

Search By:

  • National ID number, OR
  • Passport number, OR
  • ID number

Check Customer Existence › Request Body

​string

​string
